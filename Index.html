מי אני?

אני יונתן, מפתח תוכנה עם גישה מערכתית ותשוקה לבניית מוצרים אמיתיים. התחלתי מפיתוח משחקים וחוויות אינטראקטיביות, ובהמשך התמקצעתי בפיתוח Web עם דגש על React, JavaScript וארכיטקטורת קוד נקייה ומודולרית.

בפרויקט הדגל שלי, Audio Dojo, בניתי מערכת Web מלאה: ניהול state מורכב, עבודה עם Firebase ו־localStorage, מסכי סיכום והיסטוריה וייצוא דוחות PDF / Excel. הפרויקט משלב UX נגיש עם חשיבה מערכתית ויכולת עבודה עם דאטה.

עבדתי על פרויקטים עצמאיים וצוותיים, כולל במסגרת סטארט־אפ למשחק מובייל. הניסיון שלי כולל כניסה לקוד קיים, קריאה של דוקומנטציה, והטמעת פתרונות חדשים במערכות פעילות.

לצד הפיתוח שימשתי גם כעוזר הוראה בקורסי Unity, תפקיד שחיזק את היכולת שלי להסביר, לכתוב קוד ברור ולעבוד בצורה שיטתית ומסודרת.
